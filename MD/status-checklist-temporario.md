# Checklist Temporario - Sistema de Status

## Como Usar

Este arquivo e temporario e serve para acompanhar o andamento da implementacao.

Quando uma etapa for concluida, troque `Pendente` por `Feito`.

Quando uma etapa estiver em curso, troque `Pendente` por `Em andamento`.
